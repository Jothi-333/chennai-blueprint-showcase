# Saroja Ammal Memorial Photos

This directory is for photos of Saroja Ammal to be displayed on the home page tribute section.

## Instructions

1. Add photos of Saroja Ammal to this directory
2. Recommended photo names:
   - `saroja-portrait.jpg` - Main portrait photo
   - `saroja-family.jpg` - Family photo
   - `saroja-1.jpg`, `saroja-2.jpg`, etc. - Additional photos

## Photo Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 800x800px or larger
- **Quality**: High resolution for best display
- **Aspect Ratio**: Square (1:1) or portrait (3:4) works best

## Usage

These photos will be displayed in the hero section of the home page as a tribute to Saroja Ammal.

The home page will automatically display photos from this directory in a beautiful gallery layout.
